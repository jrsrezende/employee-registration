# 👨‍💼 Sistema de Cadastro de Funcionários

## 📋 Descrição

Este projeto é uma aplicação simples em Java para cadastrar funcionários, coletando informações como nome, CPF, matrícula e salário. Os dados são exibidos no console e exportados para um arquivo.

## 🛠️ Tecnologias

- Java 
- Swing (JOptionPane) 

## ⚙️ Como executar

1. Clone o repositório: git clone https://github.com/seu-usuario/seu-repositorio.git
2. Abra o projeto no IntelliJ IDEA.
3. Execute a classe `Main` (`src/application/Main.java`).

## 📝 Funcionalidades

- Cadastro de funcionário via interface gráfica simples.
- Exportação dos dados para arquivo.
- Exibição dos dados no console.
